# Project Title

A simple Express application built with TypeScript.

## Description

This project is a basic Express server that responds with a JSON message. It serves as a starting point for building more complex applications using TypeScript and Express.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd backend
   ```

3. Install the dependencies:
   ```
   npm install
   ```

## Usage

To start the server, run the following command:
```
npm start
```

The server will be running on `http://localhost:3000`.

## Endpoints

- `GET /`: Responds with a JSON message: `{ "message": "Hello, World!" }`
